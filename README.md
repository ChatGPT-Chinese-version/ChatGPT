## 什么是ChatGPT镜像网站？
镜像网站是指将原始网站的内容复制并放置在另一服务器上的网站。这个概念通常应用于提供备用访问途径，为主站点的繁重流量提供缓解。一般来说，镜像网站会更新以保持与原始网站相同的内容，但这个更新的频率可能因镜像站点的设定不同而不同。

## 为什么选择ChatGPT镜像网站？
因为官网的ChatGPT在国内是无法访问的，尽管有些小伙伴通过魔法的手段成功访问了，但是还需要用国外的信用卡跟手机卡进行账号注册，所以步骤很麻烦，哪怕你真的成功注册到账号并开通了会员，也会随时面临官网的封号，所以总的来说，不止麻烦多，而且成本也高，所以选择国内镜像网站会帮助自己减少很多麻烦。
